# HOWTO - GC Auto | Gold Coast

## Quick Start

### Development
```bash
cd ~/repara_alquila_con_ia
npm run dev
```

### Deploy to Vercel
```bash
vercel --prod
```

## Common Tasks

### Update Fleet
Edit `fleet` array in `src/app/page.tsx`

### Update Services
Edit `services` array in `src/app/page.tsx`

### Add FAQ
Add to `faqs` array in `src/app/page.tsx`

### Update SEO
Edit `metadata` in `src/app/layout.tsx`

## Troubleshooting

### Build fails
```bash
npm run build
```

### Changes not showing
```bash
vercel --prod --force
```
